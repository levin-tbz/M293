# Webshop mit KI-gestützter Webentwicklung (M293 Projektauftrag 3)

Dieses Repository enthält den Quellcode und die Dokumentation für den Webshop, der im Rahmen des Moduls M293 entwickelt wird. Das Projekt wird ohne Frameworks (wie Bootstrap oder Tailwind) rein mit semantischem HTML, CSS Grid/Flexbox und Vanilla JavaScript entwickelt.

## Projektstruktur
- `index.html` - Startseite mit neuen/beliebten Produkten und Newsletter-Formular.
- `shop.html` - Produktkatalog mit Kategorie-Filtern (Grid-Layout).
- `detail.html` - Detailansicht mit Bild, Video, Beschreibung und Direktbestell-Formular.
- `contact.html` - Kontaktseite mit Teamvorstellung und Formular inklusive Betreff-Dropdown.
- `style.css` - Globales Design-System (Typography, Colors, Spacing, Responsive Grid).
- `app.js` - Client-seitige Logik (Kategorie-Filter, Produkt-Rendering, Formular-Validierungen).
- `data/products.json` - Datenbank für mindestens 10 Produkte.
- `docs/`
  - [styleguide.md](docs/styleguide.md) - Farbschema, Typografie und CSS-Klassen-Dokumentation.
  - [wireframes.md](docs/wireframes.md) - Layout-Skizzen für Mobile, Tablet und Desktop.
  - [ai_documentation.md](docs/ai_documentation.md) - Dokumentation der KI-Nutzung und Vergleich der Tools.

## Ausführung
Da es sich um reines HTML/CSS/JS handelt, kann das Projekt direkt im Browser geöffnet oder über einen lokalen Webserver (z. B. VS Code Live Server) gestartet werden.
